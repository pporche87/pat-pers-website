YouTube Clone Built in React

January 2017

Explored learning the power of front-end development with REACTjs by building a clone of the popular YouTube video searching web application. My components included a search bar that communicated with you YouTube API and kept its information in application state. A separate component rendered the top 5 video thumbnails of the search query, whilst the last component rendered a full-screen, playable version of the number 1 query. The ability to click the thumbnail link and render a new playable was also a key feature of the project.
